DH Parameters Calculator

        To extend the application to calculate the overall transformation matrix from the base frame to the end-effector frame, you need to:

Create a function to multiply two transformation matrices.
Add a section to display the overall transformation matrix.
Update the calculateAllTransformations function to also compute and display the overall transformation matrix.

Explanation of New Features
New Function multiplyMatrices(A, B):

Multiplies two 4x4 matrices A and B.
New Function displayOverallTransformationMatrix(matrix):

Displays the overall transformation matrix in the specified HTML table.
Updates to calculateAllTransformations:

Initializes overallMatrix to the identity matrix.
Multiplies each transformation matrix with the overall matrix to compute the overall transformation sequentially.
Displays the final overall transformation matrix.
Overall Transformation Matrix Section:

Added a section to display the overall transformation matrix under the individual link transformation sections.
Usage
Adding Links: Click the "Add Link" button to add new sections for each link.
Entering DH Parameters: Enter the DH parameters for each link.
Calculating Transformations: Click the "Calculate All" button to compute and display both the individual link transformation matrices and the overall transformation matrix from the base frame to the end-effector frame.
